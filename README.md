Real-Time Chat App using React & Socket.io
A simple real-time chat app built with React (frontend), Node.js (backend), and Socket.io. Supports multiple chat rooms and real-time messaging.

Features
- Room-based messaging system
- Real-time chat updates via Socket.io
- Frontend in React with functional components
- Backend in Node.js with Express

Tech Stack
- React
- Node.js
- Socket.io
- Express.js

How to Run
Backend
```bash
cd backend
npm install
node index.js
